博客搭建记录：用 Hexo + GitHub Pages 搭建个人博客

欢迎来到我的博客！这是我用 Hexo + GitHub Pages 搭建的个人博客，这篇文章记录了整个搭建过程。

为什么选择 Hexo

在考虑了多种方案后，选择了 Hexo 的理由：

  • 静态站点：无需服务器，GitHub Pages 免费托管
  • Markdown 写作：专注内容，无需关心 HTML
  • 中文社区活跃：遇到问题容易找到解答
  • 主题丰富：从简洁到绚丽，各种风格可选
